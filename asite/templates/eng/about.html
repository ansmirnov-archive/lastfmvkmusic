About Service

The site has only one service available at the moment - Transmitting music from Last.FM status into social network "VK.com".

News

25/07/2012

What's new:

  • Now the site has at least some sort of design so you can share the link without being ashame
  • A total refactoring of the code has been carried out, now the whole service works on Python + Django. Sources will be posted later.
  • Database Users has been moved from the platform-based storage on a more versatile and flexible Django ORM.
  • Now the server component doesn't depend on the platform and (theoretically) can be scaled.

Ideas for development:

  • Custom status format (so you can replace the default "Now playing")
  • Restoration of the previous status after listening to the music.
